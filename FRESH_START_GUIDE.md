### **METHOD 2: VERCEL (Recommended for Long-Term)**

**Step 1: Get the Files**
1. Download stoneforms folder (link above)
2. Save to Desktop
3. Unzip if it's a .zip file

**Step 2: Open Terminal**
1. Press `Command + Space`
2. Type: `terminal`
3. Press Enter

**Step 3: Navigate to Folder**
```bash
cd ~/Desktop/stoneforms
```

**Step 4: Install Vercel CLI**
```bash
npm install -g vercel
```

*Don't have npm? Install Node.js first from: https://nodejs.org*

**Step 5: Deploy**
```bash
vercel
```

Follow the prompts (just press Enter for defaults)

**✅ DONE!** You'll get a URL!
